# 📊 Employee Attrition Dashboard | Power BI

## 📌 Project Overview
This project analyzes employee attrition data using **Microsoft Power BI** to identify workforce trends and the key factors contributing to employee turnover. The interactive dashboard provides HR professionals with actionable insights to support data-driven workforce planning and retention strategies.

---

## 🎯 Project Objective
The objective of this project is to analyze employee attrition patterns and build an interactive Power BI dashboard that helps organizations understand workforce trends and make informed HR decisions.

---

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📂 Dataset
The dataset contains employee information, including demographic details, job roles, salary, education, department, years at the company, and attrition status.

---

## 📈 Key Business Questions (KPIs)

- What is the overall employee attrition rate?
- Which departments have the highest attrition?
- Which job roles experience the most employee turnover?
- How does attrition vary by age group, gender, and education?
- What is the relationship between salary and attrition?
- How does years of service impact employee retention?
- Which factors contribute most to employee attrition?

---

## ⚙️ Project Workflow

1. Imported and explored the HR dataset.
2. Cleaned and transformed data using **Power Query**.
3. Created relationships and data models for analysis.
4. Developed **DAX measures** and calculated columns to calculate HR KPIs.
5. Built interactive visualizations using charts, cards, slicers, and filters.
6. Designed a user-friendly dashboard for dynamic analysis and reporting.

---

## 🔍 Key Insights

- Identified the overall employee attrition rate across the organization.
- Analyzed attrition trends by department, job role, age group, and education.
- Evaluated the relationship between employee salary, experience, and attrition.
- Highlighted key workforce patterns to support employee retention strategies.
- Enabled interactive filtering for deeper HR analysis.

---

## ✅ Business Recommendation

Organizations should focus on departments and job roles with higher attrition by implementing employee engagement initiatives, career development programs, competitive compensation strategies, and retention plans. Continuous monitoring through interactive dashboards can help HR teams make proactive, data-driven decisions.

---

## 🚀 Project Outcome

The dashboard enables HR teams to:

- Monitor employee attrition trends.
- Track important HR KPIs.
- Analyze workforce demographics.
- Identify high-risk employee groups.
- Support strategic HR planning through data-driven insights.
